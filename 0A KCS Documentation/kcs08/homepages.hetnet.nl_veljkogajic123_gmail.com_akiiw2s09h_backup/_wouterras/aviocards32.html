AvioCards32 v1.1



If you're using Delphi, and you want to program card games, you're at the right address.
AvioCards32 is a Delphi unit with all 52 playing cards, 2 world jokers en 9 different card backs, and can be integrated into any Delphi application, designed to work under Windows 95 or 98.



AvioCards32 is a zip compressed archive, which contains the following files:

AvioCards32.pas- The Delphi unit (source code),
Example1.dpr- Sample project main file,
Unit1.dfm- Sample unit form,
Unit1.pas- Sample unit.



AvioCards32 is freeware. Click here to download. In this new version (1.1) a bug with the transparent color, that may occur in some circumstances, has been fixed, with help of David Howorth.

You can also download Pyramid here, a cardgame that uses AvioCards32.
